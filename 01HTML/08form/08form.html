form 表单侧重于向后端发送数据

主要是以 input 文本标签为主,附以 name 属性,通过form表单向后端发送数据

除了 input 标签还有大文本标签 textrea;下拉菜单标签 select

        1.type: text,password,count,date,botton,radio(单选),checkbox(多选),hidden,submit(提交),reset(重置)等
        2.name: name 设置的值必须要和后端接受数据的参数名称一致
        3.value: 显示在页面上的值(保存输入的数据),除了 type="hidden" 的除外,因为它为隐藏标签
        4.autocomplete="off" 是否自动补全
        form action="后端地址" method="提交方式"
        input没有闭标签
