1 在js中,函数在定义时有多少个参数,在函数调用时就应该传递多少个参数.
    但是如果在参数传递时,参数过多,容易导致程序错误,
    为了减少参数传递的数量我们可以考虑使用对象作为函数的参数

    注意: 传递的参数是引用传递

2
<script>
    // 传递对象给函数作为参数
    function dis(obj) {
        obj.name = 'Jerry';
    }

    // 建立对象
    var o = {
        name: 'Tom',
        age: 15,
        sex: '男'
    };

    console.log(o.name); // Tom

    dis(o); // 引用传递
    console.log(o.name); // Jerry 函数里面改了,是引用传递
</script>
