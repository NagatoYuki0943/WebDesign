一.方法使用
    1.使用关键字function定义方法
    2.方法名使用 驼峰法命名
    3.方法可以设置参数以及返回值
    function myFunction(parma1,parma2){
        return parma1+parma2;
    }

二.注意
    1.由于js是弱类型语言,不需要通过指定类型声明变量,故方法名称前不需要指定变量类型
    2.方法返回值 关键字 return 后如果不跟返回的内容,可以简单理解为中断方法执行
    3.js不存在方法重载,"重载"的方法会覆盖之前的方法
    4.从方法中获取参数还可以使用 argument 关键字,它是一个数组,课后了解
