CSS语法格式

  • CSS语法是由特征和值成对儿出现
  • 每组特征之间用英文 ; 分隔
  • 除了在元素标签的 style 属性设置外,另外两种写法必须写在 {} 内

CSS语法

        在style属性中
        div style="特征1 : 值1;特征2 : 值2; 特征3 : 值3;"

        在.css文件或者style标签中
            元素选择器{
                特征1 : 值1;
                特征2 : 值2;
                特征3 : 值3;
            }
        特征可以简单理解为标签样式的类型的代号,比如元素背景色为: "background-color"
        每个特征都会对应很多值,例如设置元素背景色为红色即为:"background-color:red"
