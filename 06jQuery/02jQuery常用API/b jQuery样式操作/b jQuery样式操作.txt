1  操作 css 方法        14
    jQuery 可以使用 css 方法来修改简单元素样式； 也可以操作类，修改多个样式。

    a. 参数只写属性名，则是返回属性值
    $('div').css('width')

    b. 参数是属性名，属性值，逗号分隔，是设置一组样式，属性必须加引号，值如果是数字可以不用跟单位和引号
    $('div').css('width', '300px') //变宽了 300 300px都行

    c. 参数可以是对象形式，方便设置多组样式。属性名和属性值用冒号隔开， 属性可以不用加引号，

        //使用对象形式修改,属性名引号可加可不加
        $('div').css({
            width: 400,
            'height': 500,
            //复合属性必须采用驼峰命名法,值不是数字需要引号
            backgroundColor: 'blue'
        })



2  设置类样式方法       15
    作用等同于以前的 classList，可以操作类样式， 注意操作类里面的参数不要加点。

    a. 添加类,类名不用加点
        $(“div”).addClass('current');

    b. 移除,里面不写参数的话删除全部类
        $(“div”).removeClass('current');

    c. 切换类 添加或删除
        $(“div”).toggleClass('current');



3  类操作与className区别        17
    原生 JS 中 className 会覆盖元素原先里面的类名。
    jQuery 里面类操作只是对指定类进行操作，不影响原先的类名。
