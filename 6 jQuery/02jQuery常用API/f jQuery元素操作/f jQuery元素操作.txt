jQuery 元素操作
    主要是遍历、创建、添加、删除元素操作

1 遍历元素      04
    jQuery 隐式迭代是对同一类元素做了同样的操作。 如果想要给同一类元素做不同操作，就需要用到遍历。

    语法1：
        $("div").each(function (index, domEle) { xxx; }）       

        a. each() 方法遍历匹配的每一个元素。主要用DOM处理。 each 每一个
        b. 里面的回调函数有2个参数：  index 是每个元素的索引号;  demEle 是每个DOM元素对象，不是jquery对象
        c. 所以要想使用jquery方法，需要给这个dom元素转换为jquery对象  $(domEle)


    语法2：
        $.each(object，function (index, element) { xxx; }）       

        a. $.each()方法可用于遍历任何对象。主要用于数据处理，比如数组，对象
        b. 里面的函数有2个参数：  index 是每个元素的索引号;  element  遍历内容



2 案例：购物车案例模块-计算总计和总额
    a. 核心思路：把所有文本框里面的值相加就是总计数量。总额同理
    b. 文本框里面的值不相同，如果想要相加需要用到each遍历。声明一个变量，相加即可
    c. 点击+号-号，会改变总计和总额，如果用户修改了文本框里面的值同样会改变总计和总额
    d. 因此可以封装一个函数求总计和总额的， 以上2个操作调用这个函数即可。
    e. 注意1： 总计是文本框里面的值相加用 val()     总额是普通元素的内容用text()  
    f. 要注意普通元素里面的内容要去掉￥并且转换为数字型才能相加



3 创建元素      05
    语法：

        var li1 = $("<li>我是后来的li1</li>");
        动态的创建了一个 <li>  



4 添加元素      05
    a. 内部添加
        element.append('内容')  
        把内容放入匹配元素内部最后面，类似原生 appendChild。

        element.prepend('内容')  
        把内容放入匹配元素内部最前面。

    b. 外部添加
        element.after(''内容'')         //把内容放入目标元素后面
        element.before(''内容'')        //把内容放入目标元素前面 

    内部添加元素，生成之后，它们是父子关系。
    外部添加元素，生成之后，他们是兄弟关系。



5 删除元素      05
    element.remove()   //删除匹配的元素（本身）
    element.empty()    //删除匹配的元素集合中所有的子节点
    element.html('')   //清空匹配的元素内容,直接将里面的内容替换为一个空的字符串

    remove 删除元素本身。
    empty() 和  html('') 作用等价，都可以删除元素里面的内容，只不过 html 还可以设置内容。



6 购物车案例模块-删除商品模块
    a. 核心思路：把商品remove() 删除元素即可
    b. 有三个地方需要删除： 1. 商品后面的删除按钮 2. 删除选中的商品 3. 清理购物车
    c. 商品后面的删除按钮： 一定是删除当前的商品，所以从 $(this) 出发
    d. 删除选中的商品： 先判断小的复选框按钮是否选中状态，如果是选中，则删除对应的商品
    e. 清理购物车： 则是把所有的商品全部删掉



7 购物车案例模块-选中商品添加背景
    a. 核心思路：选中的商品添加背景，不选中移除背景即可
    b. 全选按钮点击：如果全选是选中的，则所有的商品添加背景，否则移除背景
    c. 小的复选框点击： 如果是选中状态，则当前商品添加背景，否则移除背景
    d. 这个背景，可以通过类名修改，添加类和删除类
